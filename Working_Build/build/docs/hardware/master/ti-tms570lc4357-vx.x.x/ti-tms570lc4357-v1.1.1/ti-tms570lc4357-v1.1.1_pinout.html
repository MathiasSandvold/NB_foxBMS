4.1.2.2.3. Master TMS570 v1.1.1 Pinout

Important

The following description only applies for the BMS-Master Board hardware version v1.1.1.

Hint

All connector pinouts described below follow the Convention for Molex Micro-Fit 3.0 Connector Pin Numbering, Convention for Samtec Edge Rate connectors, Convention for Mictor Pin Numbering or Convention for Wuerth WR-MJ connector.

This section describes the pinout of all connectors on the BMS-Master Board v1.1.1.
