6.2.2. Interface MC33664 v1.0.2

6.2.2.1. Overview

Important

The following description only applies for the MC33664-based BMS-Interface Board hardware version 1.0.2.

Hint

All connector pinouts described below follow the Convention for Molex Micro-Fit 3.0 Connector Pin Numbering and Convention for Samtec Edge Rate connectors.

The BMS-Interface Board is used to enable communication between the BMS-Master Board and the BMS-Slave Boards.
